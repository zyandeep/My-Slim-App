Once, the application is insalled, the user needs to create an account to log into the app. The registration process is of two types:
  • Registration with email
  • OTP(One Time Password) Login

During email registration, the user needs to provide his/her email address, a secure password along with first name and last name. This email address will be used to login later.

In case of OPT login, the user has to provide his/her phone number. An OTP is sent to the phone and the app automatically verifies the opt. Once everyting is verified, an account is created and then user can login.

In case the user wants to log out, there's a "Log out" option at the bottom of the app's Side Navigation Menu.
