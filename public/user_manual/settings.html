The "Settings" screen of the application let users customise the app according to his/her prerefence. There are basically two types of customizations:
  • Changing the app's default language
  • Changing the app's default theme

As of now, the app supports two languages: English, which is the default language of the app, and Assamese. The "Language" option of the Settings screen lets users change app's language.

In the same manner, there's an option, named "Theme", in the Settings screen for changing app's theme. The app's default theme is "Light". However, a "Dark" theme has been provided to relieve users' eyes in case they are using the app in a low light, dark enevironment.
